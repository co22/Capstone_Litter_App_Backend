How to connect to the login sqlite database:

1. import sqlite3

2. conn sqlite3.connect('login.db')

3. c = conn.cursor()

4. Perform commands
- i.e. c.execute("INSERT INTO user VALUES (x, y))
- This inserts the specified values into the user table
- c.execute("INSERT INTO user (username) VALUES ('Test')") 
- This inserts a username into the user table (id should be generated automatically)

5. conn.commit()
- Save the changes.

6. conn.close()
- Closes the connection.

Note: view the database using VS code extensions such as SQLite Viewer
